# TAf UI/UX Review Skill

Use this skill when the task modifies templates, CSS, or any user-facing text.

## What I do

Review student-facing UI for readability, mobile fit, French clarity, and
consistency with the navy/white/light-blue design system.

## Checklist

- [ ] All student-facing text is in simple, clear French
- [ ] Form labels are unambiguous for lycée students
- [ ] Error messages explain what to do, not just what is wrong
- [ ] Buttons and links have clear action text
- [ ] Layout works on mobile (320px+ width)
- [ ] No horizontal scroll on narrow screens
- [ ] Touch targets are at least 44x44px on mobile
- [ ] Color is not the only differentiator for states
- [ ] Header logo/brand is correctly displayed
- [ ] Form fields are aligned consistently in grids
- [ ] Success page clearly confirms submission

## When to use me

Load during implementation and before final review.

## Hard stops

- French text is awkward, incorrect, or above lycée reading level
- Mobile layout breaks form usability
- Design diverges from navy/white/light-blue palette without reason

## Expected output

- Specific UI findings with file/line references
- Verdict: APPROVE / REQUEST_CHANGES / BLOCK
