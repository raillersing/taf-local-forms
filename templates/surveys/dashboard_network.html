Si les élèves ne voient pas la page

  • Vérifie que le téléphone et l'ordinateur sont sur le même Wi-Fi ou hotspot.
  • Vérifie que l'adresse ci-dessus est bien l'IP de l'ordinateur (pas une IP WSL ou Docker).
  • Vérifie le pare-feu Windows : autorise Docker ou le port {{ port }}.
  • Vérifie l'absence d'AP isolation / client isolation sur le point d'accès Wi-Fi.
  • Si tu utilises WSL2 avec NAT, un portproxy (netsh) peut être nécessaire.
